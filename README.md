# 📊 Gestión de Ingresos

> Aplicación Android para registrar **movimientos monetarios** de forma simple y clara.

Pensada para **uso personal** y para personas que necesitan llevar un control básico de su dinero sin configuraciones complejas.

---

##  Funcionalidades

-  Registro de **ingresos**
-  Registro de **egresos**
-  Asociación de cada movimiento con:
   - Fecha
   - Tipo de pago
   - Motivo
-  Visualización **total** de registros
-  Exportación de datos a **Excel** (aplicación Microsoft Excel Spreadsheets **NECESARIA**)
